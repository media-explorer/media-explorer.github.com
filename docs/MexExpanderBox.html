box :

 The expander box

primary :

 TRUE to set the limit on the primary child, FALSE for the secondary.

max_width :

 Maximum width for the child in the position to be limited to or -1 to use the natural width.

max_height :

 Maximum height for the child in the position to be limited to or -1 to use the natural height.

Property Details

The "close-on-unfocus" property

  "close-on-unfocus"         gboolean              : Read / Write

Close the expander when it loses focus.

Default value: TRUE

The "expand" property

  "expand"                   gboolean              : Read / Write

Expand into allocated space.

Default value: FALSE

The "grow-direction" property

  "grow-direction"           MexExpanderBoxDirection  : Read / Write

Expand direction of the secondary child.

Default value: MEX_EXPANDER_BOX_DOWN

The "important" property

  "important"                gboolean              : Read / Write

Whether the centre child should be displayed, regardless of focus.

Default value: TRUE

The "important-on-focus" property

  "important-on-focus"       gboolean              : Read / Write

Mark the box as important when it receives focus.

Default value: FALSE

The "open" property

  "open"                     gboolean              : Read / Write

Whether the expander are open.

Default value: FALSE

The "open-on-focus" property

  "open-on-focus"            gboolean              : Read / Write

Open the expander when it receives focus.

Default value: FALSE

The "primary-child" property

  "primary-child"            ClutterActor*         : Read / Write

Primary child of the widget.

The "secondary-child" property

  "secondary-child"          ClutterActor*         : Read / Write

Child that expands out of the widget.

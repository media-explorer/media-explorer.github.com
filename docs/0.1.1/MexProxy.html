Description

MexProxy can also be used to limit the number of content items in the model which are translated to GObjects (see mex_proxy_set_limit()), and reorder the content in a model (by changing the start point from which items are added: see mex_proxy_start_at()).

 Whether to loop back to the beginning of the associated MexModel's content items once the last item is reached; if TRUE, content items before the start_at_content will be added once the last of the model's content items is reached; if FALSE, only content items from start_at_content to the last of the model's content items are added

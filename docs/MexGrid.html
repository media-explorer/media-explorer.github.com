Property Details

The "stride" property

  "stride"                   gint                  : Read / Write

Amount of widgets to pack horizontally. 0 = Automatic, based on tile width.

Allowed values: >= 0

Default value: 0

The "tile-height" property

  "tile-height"              gfloat                : Read / Write

Minimum height to set on new tiles.

Allowed values: >= -1

Default value: 202

The "tile-width" property

  "tile-width"               gfloat                : Read / Write

Minimum width to set on new tiles.

Allowed values: >= -1

Default value: 360

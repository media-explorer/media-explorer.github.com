Media Explorer

Introduction

Media Explorer is a media centre application for Linux (including MeeGo).

It leverages existing libraries (GUPnP, Grilo, Tracker, GStreamer) to find, index and play local and remote media. The application (interface coded in Mx and Clutter) enables browsing, searching, and viewing of found media.

Features

  • Smooth, intuitive animated interface for media browse, search and playback.

  • Support for media discovery and streaming on local filesystem and connected USB drives.

  • Support for media discovery and streaming from UPnP sources.

  • Configurable key binding, enabling use with standard remote control units.
